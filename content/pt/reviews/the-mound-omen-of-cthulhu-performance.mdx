<Section icon={Cpu} title="Requisitos de Sistema e Benchmarks de Hardware">

Para rodar o jogo com eficiência, seu hardware deve atender aos padrões modernos de 2026. Embora o jogo seja escalável, os ambientes de "The Mound" utilizam uma geometria pesada que pode sobrecarregar GPUs mais antigas. <Section icon={Target} title="Guia de Otimização Passo a Passo">

Siga estes passos para garantir que seu PC esteja totalmente otimizado para os horrores sobrenaturais que o aguardam em The Mound.

<Steps>
  <Step num="1" title="Atualize os Drivers de Vídeo">
    Baixe os drivers Game Ready mais recentes da NVIDIA ou AMD. Esses drivers frequentemente contêm otimizações específicas para o motor de Omen of Cthulhu que podem resolver problemas de oscilação de imagem (flickering) no lançamento.
  </Step>
  <Step num="2" title="Configure o Plano de Energia do Windows">
    Defina o Plano de Energia do Windows para "Alto Desempenho" e certifique-se de que o "Modo de Jogo" está ATIVADO nas configurações do sistema para priorizar os ciclos de CPU para o jogo.
  </Step>
  <Step num="3" title="Configuração de Escalonamento no Jogo">
    Vá até as Configurações de Vídeo e ative o DLSS/FSR. Para telas 4K, o perfil "Qualidade" ou "Equilibrado" oferece os melhores resultados. Ative o "Reflex Low Latency" se estiver disponível.
  </Step>
  <Step num="4" title="Ajuste Fino de Sombras e Névoa">
    Reduza a "Névoa Volumétrica" para o Médio. Esta configuração simula a névoa densa de The Mound, mas é extremamente pesada para as unidades de processamento da GPU.
  </Step>
</Steps>

<Callout type="info" title="Compilação de Shaders">
Na primeira inicialização, o jogo irá compilar os shaders. Não pule este processo, pois fazer isso causará travamentos severos durante os primeiros 30 minutos de gameplay.
</Callout>

</Section>

<Section icon={Shield} title="Solução de Problemas Avançada e Estabilidade">

Mesmo com o melhor hardware, problemas de desempenho podem surgir devido a conflitos de software ou peculiaridades específicas do motor gráfico.

| Problema | Causa Potencial | Solução |
| :--- | :--- | :--- |
| **Micro-stuttering** | Aplicativos em Segundo Plano | Desative overlays (Steam, Discord). |
| **CTD (Crash para a Área de Trabalho)** | Overclocking | Reverta a GPU/CPU para as frequências padrão (stock). |
| **Baixo Uso de GPU** | Gargalo de CPU | Aumente a resolução ou ative o DLDSR. |
| **Texturas Piscando** | Conflito de Drivers | Faça uma instalação limpa de drivers usando o DDU. |

<CardGrid cols={3}>
  <Card title="SSD vs HDD">
    Instalar em um SSD NVMe reduz os tempos de carregamento de 45 segundos para menos de 5 segundos e elimina os engasgos de transição de cenários.
  </Card>
  <Card title="DirectX 12">
    Certifique-se de que o DX12 está ativado para uma melhor utilização de CPUs multi-core, o que é crítico para o processamento de IA do jogo.
  </Card>
  <Card title="V-Sync">
    Desative o V-Sync do jogo se estiver usando um monitor G-Sync ou FreeSync para reduzir o atraso de entrada (input lag).
  </Card>
</CardGrid>

<Callout type="warning" title="Vazamento de Memória">
Alguns jogadores relatam vazamentos de memória após mais de 4 horas de jogo contínuo. Se você notar o FPS caindo com o tempo, reiniciar o jogo limpará o cache da VRAM.
</Callout>
